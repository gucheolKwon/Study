# DB 제도
: 근로자 퇴직 시 퇴직급여 (평균임금 30일분 * 계속근로연수)를 지급하기 위하여 회사가 금융상품으로 운용하는 제고
유리한경우 : 예상 임금인상률 > 기대 투자수익률

# DC(Defined Contribution) 제고
: 회사는 매년 연간 임금총액의 1/12을 정기적으로 근로자 DC계좌에 적립하고, 근로자가 직접 운영
유리한경우 : 예상 임금인상률 < 기대 투자수익률

# IRP (Individual Retriement Pension)
: 개인납입금에 대하여 세액공제 받을 수 있고, 퇴직금에 대한 퇴직소득세를 과세이연하는 계좌
- 개인납입금 : 세액공제 및 노후소득을 목적으로 스스로 샇아가는 개인연금계좌
- 퇴직급여 : 퇴직 또는 이직 시, 퇴직금을 받는 계좌

[절세 계좌 3총사]

1. IRP
- 55세 이전은 전액 해지만 가능 
- 위험자산 투자한도 70%
- ELB, 정기예금 등 원리금 지급상품 투자 가능
- 최대 900만원까지 세액공제 가능 (연금저축 600만원 포함)

2. 연금저축
- 나이 무관 -> 일부 인출 가능 (잔여 퇴직금 과세이연 가능)
- 위험자산 투자 제한 없음
- ETF, 리츠, 펀드만 가능 (원리금지급상품 불가)
- 최대 600만원까지 세액공제 가능

----- 1+2 = 연간 한도 1800만원 (세액공제한도 900만원), 연금 수령시까지 과세 이연

3. ISA (개인종합자산관리) : 하나의 증권사에서만 만들 수 있음. 의무보유기간 3년이고, 해지 후 재가입하면서 세액공제를 받을 수 있음
- 한도는 연간 2천만원 최대 1억원까지 이월 납입 가능 
- 직전 3년간 금융소득 종합과세 대상자 제외해서 가입 가능 -> 종합과세 대상자로 선정됐다면 만기를 길게 가져가도록
- 의무보유 3년, 3년 ~ 80년 만기 절세계좌 (만기3개월 전부터 만기일 전일까지 연장가능)
- 투자 상품 : 예금(신탁형만 가능), 채권, 펀드, ELS, 국내주식 // 운용불가상품 : 해외주식, 해외채권, 사모펀드
- 과세 혜택 : 매매차익은 비과세, ISA에서만 투자손실에 대해서는 손실 상계 
            ex) 국내주식 500만원 손실, 이자수익 500 발생 -> 순손익 0 계산 
             세금이 발생하는 투자는 ISA 계좌가 유리 (ex. 국내상장 해외 ETF, 채권, 배당주 등) -> 세후 수익률이 높고, 계좌를 운용하는 전체기간동안 과세이연  

----- 3 = 연간 한도 2000만원(한도가 이월이 되니, 미리 만들어놓는게 좋음) (비과세혜택 200만원, 초과분은 9.9% 분리과세), 연금계좌로 이전 시, 추가세액공제 최대 300만원

[ISA + 연금 계좌를 활용한 절세 자산 키우기]
예시) 
ISA 에서 원금 : 2000 / 2000 / 2000 // 순 손익  500 = 200만원 비과세, 300만원은 29.7만원 세금 
3년 만기 해지 후 4000만원 연금저축으로 전환 시, 300만원 세액공제, 3700만원 비과세 인출 가능, 2000만원 ISA 입금


[절세 계좌 종합 전략]
- 연금저축 1 (회사지원) : 회사지원금 + 월급 포함 600만원 채우기 = 79.2만원 세금 환금 
                        장기자금 운용 포커스

- 연금저축 2 (개인추가납입) : 여유자금 개인적으로 불입. 운용하다가 추후 필요할 때 원금 출금해서 사용 (비과세)
                            ISA 만기자금 이체받아 추가 세액공제 받는 용도

- IRP : 연금저축 600 세액공제 채우고, 출금안할 여유자금 있으면 300만원 세액공제 혜택받기 (39.6만원 세금 환급)
        수수료 무료 혜택 확인하고 퇴직금도 함께 운용

- ISA : 연 2천만원까지 납입하여 국내주식, ETF 등 운용
        만기전에도 원금은 출금 가능하며, 중단기 자금 운용이 가능하다. 만기자금을 연금저축 계좌로 옮기며 추가 세액공제 획득

** 절세계좌 최적 활용 순서
1) 연금저축 활용 (600만원 한도), 2) IRP 추가 납입 (300만원 한도) 3) ISA 활용 (연 2천만원 한도) 4) ISA -> 연금계좌 (만기 후 이체)


[개인연금계좌 출금순서]
- 인출순서 : 1) 과세제외 금액 2) 퇴직금 3) 개인납입액(세액공제) + 운용수익


[개인연금 상품 안내]
- ETF (Exchange Traded Fund) : 특정 지수, 섹터, 테마 등을 추종하도록 설계된 상장지수펀드
- 펀드 : 여러 투자자들의 돈을 모아 전문가가 대신 투자하고 수익을 나눠갖는 간접투자상품
- 리츠 : REITs(Real Estate Investment Trusts) : 소액 투자자들이 돈을 모아 대형 부동산에 간접적으로 투자하고, 그 수익을 배당으로 돌려받는 금융상품

[퇴직연금 상품 안내]
- 원리금보장 상품 : (저축)은행 예금, 원리금보장 ELB, 이율보증형, 채권(만기보유) 등
- 실적배당형 : 펀드, TDF, ETF, 리츠, 채권(중도매매) 등

[왜 투자해야 되는가?]
1. 노후 빈곤과 장수 리스크
- 평균 61세에 적자로 전환되며, 65세 이후 적자 규모는 2.5억원(2010년) -> 3.5억원(2021년)으로 급증
- 기대수명은 83.5세인 반면, 최빈사망연령은 90세로 7년 가까운 괴리가 존재

2. 부족한 노후 준비
- 퇴직연금은 원리금 보장(예금성) 자산에 무려 최저 72%(IRP), 최대 95%(DB) 투자
- 퇴직연금 최근 10년 연평균 수익률은 2.1%에 불과
- 부부기준 최소 노후 생활비는 월 240만원('23년 231만원), 적정 노후 생활비는 336만원('23년 324만원) 

3. 은퇴 후 투자의 중요성
- 퇴직급여 10억원 은퇴자가 매월 세후 335만원 연금 수령할 경우, 퇴직급여 고갈시점은
  - 투자 안할 경우 21년, 연 수익률 3.5% 시 30년, 4.5% 시 33년, 5.5% 시 36년 소요
- 매월 세후 450만원을 30년간 연금 수령하기 위해서는, 연 3% 수익률 가정 시 은퇴자산 18억원 필요

[어디에 투자해야 되는가?]
- 장기 수익률 압도적 1등 = "미국"
  -> 최근 15년 미국 S&P 500과 나스닥100 지수는 각각 460%, 1,131% 누적수익률
- 우리나라는? -> 코스피 96%, 코스닥 35%
- S&P500 지수 35년 연평균 상승률 10.7%
- "Time in the Market, not timing the market"
  -> S&P500 적립 투자 = 성공투자의 공식
  : 20년간 매월 100만원 적립 가정 시, 누적원금 2.4억원, 평가액 7.9억원
